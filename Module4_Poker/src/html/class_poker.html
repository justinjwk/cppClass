◆ pairCounter()

int Poker::pairCounter ( int  rankIn[NUMBER_OF_HAND])

This method counts how many pairs in hand If there are 4 same rank cards, it returns FOUR_OF_A_KIND If there are 3 same rank cards with one pair, it returns FULL_HOUSE If there are only 3 same rank cards, it returns THREE_OF_A_KIND If there are more than one one-pair, it returns TWO_PAIR If there are only one one-pair, it returns ONE_PAIR
